SmartShoe

My project for BlueStamp Engineering is to create a SmartShoe that tracks user data, such as heart rate, location, foot pressure, etc., and stores that data into an SD card or sends it to the user's cell phone via Bluetooth. This project will be using an Arduino Lilypad and I have explored many different sensors and circuits to ensure my understanding of the components of my final project. For more information, please feel free to refer to my website on the BlueStamp Engineering Website!

JavaRun

JavaRun was my final project for my AP Computer Science class in Junior year and this was really a pivot for my computer science background because it was the first time I was able to step out of my comfort zone and create something without any instruction. The project was to create whatever we would like, so my partner and I decided to make an infinite 2D runner. We challenged ourselves to make something new with a touch of the classics. This game was completely built using Java, which proved to be a challenge during many stages of the project, but the result was definitely worth it, as the game is compatible with Apple and Windows devices. Below you can find my presentation, .jar file, and GitHub with all the code and documentation!
