JavaRun

JavaRun was my final project for my AP Computer Science class in Junior year and this was really a pivot for my computer science background because it was the first time I was able to step out of my comfort zone and create something without any instruction. The project was to create whatever we would like, so my partner and I decided to make an infinite 2D runner. We challenged ourselves to make something new with a touch of the classics. This game was completely built using Java, which proved to be a challenge during many stages of the project, but the result was definitely worth it, as the game is compatible with Apple and Windows devices.
